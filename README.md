# Portfolio Landing Page - Data Engineer

Uma Landing Page moderna e responsiva para portfólio de Engenheiro de Dados, construída com React Native e Expo.

## 🎨 Design

- **Dark Mode** com tema focado em tecnologia
- **Cores principais**: Cyan (#00d4ff) e Purple (#a855f7)
- **Tipografia**: Moderna e limpa
- **Responsivo**: Adaptado para mobile e web

## 📱 Seções

1. **Hero** - Apresentação impactante com animações
2. **Skills** - Destaque para Python, R e MySQL (DDL/Otimização)
3. **AI Workflow** - Como IA é usada para automatizar e aprimorar sistemas
4. **Projects** - Projetos em destaque com placeholders
5. **Contact** - Links para LinkedIn, GitHub e Email

## 🚀 Tecnologias

- React Native com Expo
- TypeScript
- Expo Linear Gradient
- Lucide Icons
- React Native Safe Area Context

## 📦 Instalação

```bash
# Instalar dependências
npm install

# Iniciar no navegador (web)
npm run web

# Iniciar no Android
npm run android

# Iniciar no iOS
npm run ios
```

## ⚙️ Personalização

### Informações Pessoais

Edite os seguintes arquivos para adicionar suas informações:

1. **Hero Section** (`src/components/Hero.tsx`)
   - Nome: Procure por "Seu Nome"
   - Estatísticas: Anos de experiência, projetos, etc.

2. **Skills Section** (`src/components/Skills.tsx`)
   - Níveis de proficiência
   - Tags e tecnologias

3. **Projects Section** (`src/components/Projects.tsx`)
   - Adicione seus projetos reais
   - Links para GitHub/Demo

4. **Contact Section** (`src/components/Contact.tsx`)
   - Links do LinkedIn, GitHub
   - Email de contato
   - Localização

### Cores

Edite `src/styles/theme.ts` para personalizar as cores do tema.
